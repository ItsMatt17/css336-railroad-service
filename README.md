
## Dependencies

- Apache Tomcat v8.0
- Java 8.0+
- Eclipse Java EE
- MySQL Server


## Run Locally

Clone the project

```bash
  git clone https://github.com/ItsMatt17/css336-railroad-service
```

Go to the project directory

```bash
  cd css336-railroad-service
```

 Setup server MySQL and schema

```
  src/main/resources/db.sql

```
